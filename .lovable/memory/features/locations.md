## Purpose
Group buildings by area for search and reporting (e.g. Petaling Jaya).

## Fields
- Location Name

## Table Columns
No., Location, Buildings Count, Units Count, Rooms Count, Actions

## Actions
Add, View, Edit, Delete

## UI
Small simple modal (sm) for Add/Edit. Do not overdesign.
