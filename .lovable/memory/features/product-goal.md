## Purpose
Room rental operations portal centralizing: building access, unit/room/carpark inventory, tenant data, booking/move-in/move-out workflows, agent deals & earnings, admin payouts, user management, announcements, activity log.

## Problems to Solve
1. Building/access info scattered — agents keep asking same questions
2. Room/carpark availability tracked in Google Sheets, easily outdated
3. Agents submit booking proof after hours, staff forget to update room status
4. Tenant info collected inconsistently (WhatsApp/Google Form)
5. Bookings, move-ins, earnings manually tracked and easy to miss
6. Admin and agent need different workflows after booking approval
7. UI must be highly standardized — same action always looks/behaves the same
8. Super Admin needs full audit trail of critical changes
