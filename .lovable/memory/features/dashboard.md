## Admin Dashboard
**Purpose**: Surface urgent operational actions, reduce missed updates and manual follow-up.

**Widgets**:
- Submitted Bookings waiting for review
- Move-ins Submitted waiting for review
- Rooms with Available Soon date reached but not confirmed ready
- Carparks with Available Soon date reached but not confirmed ready
- Pending Payout items
- Recently changed records
- Quick actions

**Quick Actions**: Add Booking, Add Tenant, Add Unit, Review Bookings, Review Move-ins, Generate Payout

## Agent Dashboard
**Purpose**: Show agent exactly what to do next.

**Widgets**:
- Available Rooms count
- My Submitted Bookings
- Move-ins Ready for Move-in
- Move-ins Submitted waiting for admin
- My Deals count
- Earnings summary
- Active announcements

**Quick Actions**: Browse Available Rooms, Submit Booking, Submit Move-in
